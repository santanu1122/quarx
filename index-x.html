Welcome to Quarx

Introduction

Quarx is a powerful and yet simplistic framework for building dynamic-modular web applications.

Built on the power of CodeIgniter it provides a way to initiate a web application in minutes or add an administration layer to an existing web application.

Requirements

  • PHP 5.1.6+
  • Apache rewrite_module

Installation

in order to install Quarx, please add /install to the URL.

Files to Modify After Installation:

/application/errors/error_404.php - You need to set any details you prefer in this such as the email address you want the error report sent to.
